# SC3000 Lab 1

Team: greedy for A+

## Overview

This repository contains solutions for SC3000 Lab Assignment 1:

- Part 1: Graph Search Algorithms (Dijkstra, UCS with energy constraint, A\*)
- Part 2: Grid World MDP and Reinforcement Learning (Value Iteration, Policy Iteration, Monte Carlo Control, Q-Learning)

## Project Structure

- main.py: Runs Part 1 and Part 2
- part1.py: Graph search algorithms
- part2.py: Grid world MDP and RL algorithms
- G.json, Coord.json, Dist.json, Cost.json: Graph data files
- diagrams: Helper scripts for plotting and analysis

## Requirements

- Python 3.8+
- numpy

## How to Run

Run the full lab (Part 1 and Part 2):

```bash
python main.py
```

Run only Part 1:

```bash
python part1.py
```

Run only Part 2:

```bash
python part2.py
```
